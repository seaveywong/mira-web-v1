生效日期：2026 年 6 月 26 日

1. 服务说明

本平台提供 Meta（Facebook）广告投放管理工具，帮助用户通过 Meta Ads API 创建、监控、管理广告账户及投放活动。

2. 用户责任

用户须确保通过本平台操作广告账户的行为符合 Meta 广告政策及适用法律。用户对自己设置投放规则、预算阈值及自动化操作的结果承担全部责任。

3. 授权与 Token

用户通过 Meta OAuth 或系统用户授权本平台访问广告账户。用户可随时在 Facebook 设置中撤销授权。撤销后本平台将无法继续访问对应广告账户。

4. 服务可用性

我们尽合理努力维持服务稳定运行，但不保证服务不中断。因 Meta API 变更、网络故障、不可抗力等导致的服务中断不承担赔偿责任。

5. 责任限制

在法律允许的范围内，本平台对因使用或无法使用本服务导致的间接损失（包括但不限于广告消耗损失、利润损失）不承担责任。

6. 终止

我们保留因违反本条款或适用法律而终止服务访问的权利。用户可随时停止使用本服务并要求删除账户数据。

7. 条款变更

我们可能更新本条款，重大变更将通过系统通知用户。继续使用即表示接受修订后的条款。
