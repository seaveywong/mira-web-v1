Mira Local Token Bridge

Install:
1. Open chrome://extensions
2. Enable Developer mode
3. Load unpacked and select this folder, or drag the zipped package after extracting it
4. In Mira > Token 管理, create a local-token binding code
5. Paste the Mira URL and binding code into the extension
6. Paste the local Meta access token and click 立即上报

The extension does not read Facebook cookies or hidden browser session data.
It only sends the token explicitly provided by the user.
